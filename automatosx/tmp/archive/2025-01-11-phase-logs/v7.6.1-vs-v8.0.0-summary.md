### P0 - Critical Gaps

1. **Interactive CLI Mode** ‚ùå
   - v7.6.1: `ax cli` launches ChatGPT-style REPL
   - v8.0.0: Only standard commands, no REPL
   - **Impact**: Experts-only, not beginner-friendly

2. **Spec-Kit Workflow Generation** ‚ùå
   - v7.6.1: `ax spec create`, `ax gen plan/dag/scaffold/tests`
   - v8.0.0: Manual YAML only
   - **Impact**: Cannot auto-generate workflows from NL descriptions

3. **Natural Language Commands** ‚ùå

   - v8.0.0: Must use exact syntax: `ax workflow run workflows/security-audit.yaml`
   - **Impact**: Must memorize commands
