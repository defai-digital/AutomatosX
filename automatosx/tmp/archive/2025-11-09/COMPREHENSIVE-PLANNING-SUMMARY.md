## Key Architectural Decisions

### 1. Hybrid ReScript + TypeScript
- **ReScript**: Core logic, state machines, deterministic workflows
- **TypeScript**: Services, CLI, integrations, UI
- **Rationale**: Type safety + JavaScript ecosystem access

### 2. SQLite with FTS5
- **Choice**: SQLite over PostgreSQL
- **Rationale**: Embedded, fast, FTS5 for search, perfect for local-first

### 3. ML Semantic Search
- **Model**: all-MiniLM-L6-v2 (384 dimensions)
- **Vector DB**: Faiss with HNSW index
- **Rationale**: Balance of accuracy, speed, and resource usage

### 4. Kubernetes for Scale
- **Orchestration**: Kubernetes
- **Package Management**: Helm
- **Rationale**: Industry standard, auto-scaling, high availability

### 5. LSP for Editor Integration
- **Protocol**: Language Server Protocol
- **Rationale**: Works with all major editors (VS Code, Vim, Emacs)
