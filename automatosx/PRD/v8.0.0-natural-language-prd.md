## Executive Summary

Natural Language Interface enables users to execute any AutomatosX command using conversational language instead of precise command syntax. This feature eliminates the command memorization barrier and makes AutomatosX accessible to users who prefer natural dialogue over CLI commands.

**Impact:**
- **Accessibility:** Zero learning curve for new users
- **Discoverability:** Natural questions reveal capabilities
- **Productivity:** Faster execution through natural expression
- **Error Reduction:** No syntax errors, only intent clarification

**Success Criteria:**
- Accept natural language for all 15+ CLI commands
- >90% intent classification accuracy
- >85% parameter extraction accuracy
- <2s latency for simple commands
- <5s latency for complex commands with LLM fallback
- Graceful degradation to command suggestions

## Problem Statement

### Current State (v8.0.0)

Requires exact command syntax:
```bash
# Must remember exact commands and flags
ax workflow run workflows/security-audit.yaml --config config.json
ax memory search "authentication logic" --limit 10
ax agent list --filter backend --format json
ax gen plan workflows/test.yaml --format text
```

**Barriers:**
- Must memorize 15+ commands
- Must remember flag names and order
- Syntax errors frustrate beginners
- No natural way to explore capabilities
- Cognitive load of remembering exact syntax

### Target State (v7.6.1 Parity)

Natural language commands (90% coverage):
```bash
# Natural expressions → Exact commands
$ ax "run a security audit"
→ ax workflow run workflows/security-audit.yaml

$ ax "show me all backend agents"
→ ax agent list --filter backend

$ ax "find authentication logic in my codebase"
→ ax memory search "authentication logic"

$ ax "what's the execution plan for my workflow?"
→ ax gen plan workflows/<inferred>.yaml

$ ax "list recent conversations"
→ ax memory history --limit 10

$ ax "check system status"
→ ax status --verbose
```

**Intent Clarification:**
```bash
$ ax "run security thing"

❓ Which security workflow?
   1. security-audit.yaml
   2. security-scan.yaml
   3. vulnerability-check.yaml

Your choice (1-3): 1

✅ Running: ax workflow run workflows/security-audit.yaml
```

### Core Components

#### 1. IntentClassifier (`src/nlp/IntentClassifier.ts`)

**Responsibilities:**
- Classify user intent (which command to execute)
- Fast pattern matching for common phrases
- LLM fallback for ambiguous input
- Confidence scoring

**Intents (15 CLI Commands):**
```typescript
export type Intent =
  | 'workflow_run'
  | 'workflow_list'
  | 'memory_search'
  | 'memory_history'
  | 'agent_list'
  | 'agent_info'
  | 'gen_plan'
  | 'gen_dag'
  | 'gen_scaffold'
  | 'gen_tests'
  | 'spec_create'
  | 'status'
  | 'config'
  | 'telemetry'
  | 'help'
  | 'unknown';

export interface ClassificationResult {
  intent: Intent;
  confidence: number; // 0.0-1.0
  reasoning?: string;
  alternatives?: Array<{ intent: Intent; confidence: number }>;
}

export class IntentClassifier {
  private patterns: IntentPattern[] = [
    // Workflow commands
    { regex: /^(run|execute|start)\s+(workflow|wf)/i, intent: 'workflow_run', confidence: 0.95 },
    { regex: /^(run|execute)\s+(.+\.yaml|.+\.yml)/i, intent: 'workflow_run', confidence: 0.90 },
    { regex: /^(list|show|display)\s+workflows?/i, intent: 'workflow_list', confidence: 0.95 },

    // Memory commands
    { regex: /^(search|find|query|look\s+for)\s+/i, intent: 'memory_search', confidence: 0.85 },
    { regex: /^(show|list|display)\s+(recent|last|history)/i, intent: 'memory_history', confidence: 0.90 },

    // Agent commands
    { regex: /^(list|show|display)\s+agents?/i, intent: 'agent_list', confidence: 0.95 },
    { regex: /^(show|describe|info)\s+agent\s+/i, intent: 'agent_info', confidence: 0.90 },

    // Generation commands
    { regex: /^(generate|gen|create)\s+(plan|execution\s+plan)/i, intent: 'gen_plan', confidence: 0.95 },
    { regex: /^(generate|gen|create)\s+(dag|graph|dependency)/i, intent: 'gen_dag', confidence: 0.95 },
    { regex: /^(generate|gen|create)\s+(scaffold|project|structure)/i, intent: 'gen_scaffold', confidence: 0.95 },
    { regex: /^(generate|gen|create)\s+tests?/i, intent: 'gen_tests', confidence: 0.95 },

    // Spec-kit
    { regex: /^(create|generate|make)\s+(spec|workflow)\s+/i, intent: 'spec_create', confidence: 0.90 },

    // System commands
    { regex: /^(show|check|display)\s+(status|health)/i, intent: 'status', confidence: 0.95 },
    { regex: /^(show|display|get)\s+config/i, intent: 'config', confidence: 0.95 },
    { regex: /^(show|display)\s+telemetry/i, intent: 'telemetry', confidence: 0.95 },
    { regex: /^(help|\?|what|how)/i, intent: 'help', confidence: 0.80 }
  ];

  constructor(private providerRouter: ProviderRouterV2) {}

  async classify(input: string): Promise<ClassificationResult> {
    // Fast path: pattern matching
    for (const pattern of this.patterns) {
      if (pattern.regex.test(input)) {
        return {
          intent: pattern.intent,
          confidence: pattern.confidence
        };
      }
    }

    // Slow path: LLM classification
    return this.classifyWithLLM(input);
  }

  private async classifyWithLLM(input: string): Promise<ClassificationResult> {
    const response = await this.providerRouter.request({
      messages: [
        {
          role: 'system',
          content: `You are an intent classifier for AutomatosX CLI.

Available intents:
- workflow_run: Run a workflow file
- workflow_list: List available workflows
- memory_search: Search conversation memory
- memory_history: Show conversation history
- agent_list: List available agents
- agent_info: Show agent details
- gen_plan: Generate execution plan
- gen_dag: Generate dependency graph
- gen_scaffold: Generate project structure
- gen_tests: Generate test suite
- spec_create: Create workflow from description
- status: Show system status
- config: Show configuration
- telemetry: Show telemetry data
- help: Show help
- unknown: Cannot determine intent

Return JSON only:
{
  "intent": "...",
  "confidence": 0.0-1.0,
  "reasoning": "...",
  "alternatives": [{"intent": "...", "confidence": 0.0-1.0}]
}`
        },
        {
          role: 'user',
          content: input
        }
      ],
      temperature: 0.1
    });

    return JSON.parse(response.content);
  }
}
```

#### 2. EntityExtractor (`src/nlp/EntityExtractor.ts`)

**Responsibilities:**
- Extract command parameters from natural language
- Identify files, filters, limits, formats
- Handle ambiguous references
- Provide confidence scores

**Entities:**
```typescript
export interface Entity {
  type: EntityType;
  value: string;
  confidence: number;
  startIndex: number;
  endIndex: number;
}

export type EntityType =
  | 'workflow_file'
  | 'agent_name'
  | 'query_text'
  | 'filter'
  | 'limit'
  | 'format'
  | 'output_path'
  | 'config_file'
  | 'safety_level';

export interface ExtractionResult {
  entities: Entity[];
  parameters: Record<string, unknown>;
  confidence: number;
  ambiguities: Ambiguity[];
}

export interface Ambiguity {
  parameter: string;
  candidates: string[];
  question: string;
}

export class EntityExtractor {
  private entityPatterns: EntityPattern[] = [
    // Workflow files
    { regex: /([a-z0-9-]+\.ya?ml)/i, type: 'workflow_file' },
    { regex: /workflows?\/([a-z0-9-]+)/i, type: 'workflow_file' },

    // Agent names
    { regex: /agent\s+([a-z]+)/i, type: 'agent_name' },
    { regex: /(backend|frontend|security|quality|testing|devops|architect|product|cto|ceo)/i, type: 'agent_name' },

    // Query text (quoted or after "find/search")
    { regex: /"([^"]+)"/i, type: 'query_text' },
    { regex: /(find|search|query|look\s+for)\s+(.+)/i, type: 'query_text', captureGroup: 2 },

    // Filters
    { regex: /(filter|where|type)\s+([a-z]+)/i, type: 'filter', captureGroup: 2 },

    // Limits
    { regex: /(limit|top|first|last)\s+(\d+)/i, type: 'limit', captureGroup: 2 },

    // Formats
    { regex: /(format|as|output)\s+(json|yaml|text|table)/i, type: 'format', captureGroup: 2 },

    // Output paths
    { regex: /(to|into|output)\s+([\/\w.-]+)/i, type: 'output_path', captureGroup: 2 },

    // Safety levels
    { regex: /(safety|risk)\s+(permissive|normal|paranoid)/i, type: 'safety_level', captureGroup: 2 }
  ];

  constructor(
    private fileSystem: FileSystemService,
    private agentRegistry: AgentRegistry
  ) {}

  async extract(input: string, intent: Intent): Promise<ExtractionResult> {
    const entities: Entity[] = [];

    // Extract entities using patterns
    for (const pattern of this.entityPatterns) {
      const matches = input.matchAll(new RegExp(pattern.regex, 'gi'));
      for (const match of matches) {
        const captureGroup = pattern.captureGroup || 1;
        const value = match[captureGroup];

        entities.push({
          type: pattern.type,
          value,
          confidence: 0.9,
          startIndex: match.index!,
          endIndex: match.index! + match[0].length
        });
      }
    }

    // Convert entities to parameters
    const parameters: Record<string, unknown> = {};
    const ambiguities: Ambiguity[] = [];

    for (const entity of entities) {
      switch (entity.type) {
        case 'workflow_file':
          const workflowPath = await this.resolveWorkflow(entity.value);
          if (workflowPath) {
            parameters.workflowPath = workflowPath;
          } else {
            ambiguities.push({
              parameter: 'workflowPath',
              candidates: await this.findSimilarWorkflows(entity.value),
              question: `Which workflow did you mean by "${entity.value}"?`
            });
          }
          break;

        case 'agent_name':
          const agent = this.agentRegistry.get(entity.value);
          if (agent) {
            parameters.agentName = agent.name;
          } else {
            ambiguities.push({
              parameter: 'agentName',
              candidates: this.agentRegistry.list().map(a => a.name),
              question: `Which agent did you mean by "${entity.value}"?`
            });
          }
          break;

        case 'query_text':
          parameters.query = entity.value;
          break;

        case 'filter':
          parameters.filter = entity.value;
          break;

        case 'limit':
          parameters.limit = parseInt(entity.value, 10);
          break;

        case 'format':
          parameters.format = entity.value;
          break;

        case 'output_path':
          parameters.outputPath = entity.value;
          break;

        case 'safety_level':
          parameters.safetyLevel = entity.value;
          break;
      }
    }

    return {
      entities,
      parameters,
      confidence: this.calculateConfidence(entities, ambiguities),
      ambiguities
    };
  }

  private async resolveWorkflow(partial: string): Promise<string | null> {
    // Try exact match
    const exactPath = `workflows/${partial}`;
    if (await this.fileSystem.exists(exactPath)) {
      return exactPath;
    }

    // Try with .yaml extension
    if (!partial.endsWith('.yaml') && !partial.endsWith('.yml')) {
      const withExt = `workflows/${partial}.yaml`;
      if (await this.fileSystem.exists(withExt)) {
        return withExt;
      }
    }

    return null;
  }

  private async findSimilarWorkflows(partial: string): Promise<string[]> {
    const files = await this.fileSystem.listFiles('workflows');
    return files
      .filter(f => f.includes(partial))
      .slice(0, 5);
  }

  private calculateConfidence(entities: Entity[], ambiguities: Ambiguity[]): number {
    if (ambiguities.length > 0) {
      return 0.6; // Low confidence with ambiguities
    }

    const avgConfidence = entities.reduce((sum, e) => sum + e.confidence, 0) / entities.length;
    return avgConfidence || 0.5;
  }
}
```

#### 3. CommandMapper (`src/nlp/CommandMapper.ts`)

**Responsibilities:**
- Map intent + parameters to exact CLI command
- Build command string with all flags
- Validate parameter combinations
- Provide command preview

**Implementation:**
```typescript
export interface MappedCommand {
  command: string;
  args: string[];
  preview: string; // Human-readable command
  confidence: number;
}

export class CommandMapper {
  private commandTemplates: Record<Intent, CommandTemplate> = {
    workflow_run: {
      command: 'workflow run',
      requiredParams: ['workflowPath'],
      optionalParams: ['config', 'iterate', 'maxIterations', 'safety'],
      buildArgs: (params) => {
        const args = [params.workflowPath];
        if (params.config) args.push('--config', params.config);
        if (params.iterate) args.push('--iterate');
        if (params.maxIterations) args.push('--max-iterations', params.maxIterations);
        if (params.safety) args.push('--safety', params.safety);
        return args;
      }
    },

    memory_search: {
      command: 'memory search',
      requiredParams: ['query'],
      optionalParams: ['limit', 'format'],
      buildArgs: (params) => {
        const args = [params.query];
        if (params.limit) args.push('--limit', params.limit);
        if (params.format) args.push('--format', params.format);
        return args;
      }
    },

    agent_list: {
      command: 'agent list',
      requiredParams: [],
      optionalParams: ['filter', 'format'],
      buildArgs: (params) => {
        const args: string[] = [];
        if (params.filter) args.push('--filter', params.filter);
        if (params.format) args.push('--format', params.format);
        return args;
      }
    },

    gen_plan: {
      command: 'gen plan',
      requiredParams: ['workflowPath'],
      optionalParams: ['format', 'output'],
      buildArgs: (params) => {
        const args = [params.workflowPath];
        if (params.format) args.push('--format', params.format);
        if (params.output) args.push('--output', params.output);
        return args;
      }
    },

    // ... similar for all 15 intents
  };

  map(
    intent: Intent,
    parameters: Record<string, unknown>
  ): MappedCommand | null {
    const template = this.commandTemplates[intent];
    if (!template) {
      return null;
    }

    // Check required parameters
    const missingParams = template.requiredParams.filter(
      param => !parameters[param]
    );

    if (missingParams.length > 0) {
      throw new Error(`Missing required parameters: ${missingParams.join(', ')}`);
    }

    // Build args
    const args = template.buildArgs(parameters);

    return {
      command: template.command,
      args,
      preview: `ax ${template.command} ${args.join(' ')}`,
      confidence: 0.9
    };
  }
}
```

#### 4. ClarificationHandler (`src/nlp/ClarificationHandler.ts`)

**Responsibilities:**
- Ask clarifying questions when ambiguous
- Present choices to user
- Collect user response
- Retry classification with clarified input

**Implementation:**
```typescript
export class ClarificationHandler {
  async clarify(ambiguity: Ambiguity): Promise<string> {
    console.log(chalk.yellow(`\n❓ ${ambiguity.question}`));

    ambiguity.candidates.forEach((candidate, index) => {
      console.log(`   ${index + 1}. ${candidate}`);
    });

    const rl = readline.createInterface({
      input: process.stdin,
      output: process.stdout
    });

    return new Promise((resolve) => {
      rl.question(chalk.cyan(`\nYour choice (1-${ambiguity.candidates.length}): `), (answer) => {
        rl.close();

        const choice = parseInt(answer.trim(), 10);
        if (choice > 0 && choice <= ambiguity.candidates.length) {
          resolve(ambiguity.candidates[choice - 1]);
        } else {
          console.log(chalk.red('Invalid choice. Please try again.'));
          resolve(this.clarify(ambiguity));
        }
      });
    });
  }

  async clarifyAll(ambiguities: Ambiguity[]): Promise<Record<string, string>> {
    const clarifications: Record<string, string> = {};

    for (const ambiguity of ambiguities) {
      clarifications[ambiguity.parameter] = await this.clarify(ambiguity);
    }

    return clarifications;
  }
}
```

Add natural language handler to main CLI:

```typescript
// src/cli/index.ts
program
  .argument('[input...]', 'Natural language command')
  .action(async (input: string[]) => {
    if (input.length === 0) {
      // Show help
      program.help();
      return;
    }

    const naturalLanguage = input.join(' ');

    // Process natural language
    const nlp = new NaturalLanguageProcessor(
      providerRouter,
      agentRegistry,
      fileSystem
    );

    const result = await nlp.process(naturalLanguage);

    if (result.success) {
      console.log(chalk.green(`\n✅ Executing: ${result.command.preview}`));
      await executeCommand(result.command);
    } else {
      console.log(chalk.red(`\n❌ Could not understand: "${naturalLanguage}"`));
      console.log(chalk.yellow('Did you mean one of these?'));
      result.suggestions.forEach(suggestion => {
        console.log(`   - ${suggestion}`);
      });
    }
  });
```

## Risk Mitigation

### Risk 1: Poor Intent Classification

**Probability:** Medium
**Impact:** High (bad UX)

**Mitigation:**
- Hybrid approach: Fast patterns + LLM fallback
- Confidence thresholds: <0.7 = ask for clarification
- Continuous learning from corrections
- Telemetry to identify common failures

### Risk 2: Ambiguous Parameter Extraction

**Probability:** High
**Impact:** Medium (requires clarification)

**Mitigation:**
- Clear clarification questions with numbered choices
- File/agent fuzzy matching
- Context-aware suggestions
- Allow explicit syntax for disambiguation

### Risk 3: Performance Issues (LLM Latency)

**Probability:** Low
**Impact:** Medium (slow UX)

**Mitigation:**
- Pattern matching covers 80%+ of cases (<1s)
- LLM only for truly ambiguous input
- Cache common classifications
- Streaming responses to show progress
